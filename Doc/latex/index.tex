Программа по определению типа дорожного знака на изображении. Принимает на вход путь к директории, в которой располагаются изображения типы P\+P\+M see also \hyperlink{_controller_8h}{Controller.\+h} читает имя изображения из директории. Затем создаёт cv\+::\+Mat из изображения по считанному пути. Затем производит коррекции над полученной матрицей see also \hyperlink{_correction_8h}{Correction.\+h}. Скорректированные изображения поступают в see also \hyperlink{_search_outlines_8h}{Search\+Outlines.\+h} и там производится поиск контуров по скорректированным изображениям и их H\+S\+V значениям. Выделенные контуры поступают в see also \hyperlink{_contour_analis_8h}{Contour\+Analis.\+h}, Где выволняет контурный анализ и выявляются круги-\/овалы(O), ромбы-\/квадраты(R), триугольники(\+D). После производится расчет вероятности распознания знака.
